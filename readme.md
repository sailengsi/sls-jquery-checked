项目采用fis编写，fis官网地址：[http://fis.baidu.com/](http://fis.baidu.com/ "fis文档")   
所以你需要按照以下步骤才能运行此项目


	//克隆项目
	git clone https://github.com/sailengsi/sls-jquery-checked.git
	
	//进入项目中的dev目录
	cd sls-jquery-checked/src/dev
	
	//安装依赖
	npm install
	
	//启动Server
	npm start

**注意：npm start启动的服务，实际上是访问的src/release目录。**
